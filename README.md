# Creepers

Creepers is a OpenFrameworks game, based on MOBA genre.


### How to play

#### Commands

|Key            |Key                                |
|---------------|-----------------------------------|
|Left Button    |Shoot         
|Right Button   |Move player
|Middle Button  |Block waypoint if they are visible
|i              |Show Waypoints
|+              |Adds enemy speed
|-              |Subtracts enemy speed
|a              |Adds enemy spawn time
|s              |Subtracts enemy spawn time
